Select a format to preview a print-accurate surface. The preview and printed page share the same dimensions and padding for alignment testing.

Current assumptions

  • • US Letter base pages (8.5in × 11in)
  • • 0.25in outer margins to avoid printer clipping
  • • Consistent gutters for double-sided alignment

Print checklist

  1. Set scaling to 100% / Actual Size.
  2. Use system print dialog for accurate margins.
  3. Enable double-sided printing; flip on long edge.

Alignment checklist

  • 1 Measure outer margins: 0.25in on each edge.
  • 2 Verify cut lines pass through both sides.
  • 3 Ensure safe areas stay within the dashed inset.
  • 4 Print a test sheet before bulk jobs.
